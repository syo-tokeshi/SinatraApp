# アプリ概要
Sinatraを利用したメモアプリになります
# 著者の環境

- Bundler：2.4.1
- Ruby：3.2.0
- macOS Big Sur  バージョン11.2.3

# 実行手順

1. ローカル環境で以下のコマンドを実行
```
git clone https://github.com/syo-tokeshi/SinatraApp/tree/develop

cd SinatraApp

bundle install

bundle exec ruby app.rb
```

2. ブラウザから以下のURLにアクセスします
http://localhost:4567
